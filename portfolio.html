Egg A Luck

Conceptual design for a farming simulation game tailored for SUT Farm.

Status: Work in Progress

Overview: Developed as a Mini Project for the Computer Game Development course under Game Jam-style constraints. The project followed a Q1 Seasonal Event prompt, leading to the creation of an Easter-themed game. It is designed as a casual, fast-paced game focusing on quick gameplay loops.

Key Responsibilities:

  • Game Design: Collaborated on the Game Design Document (GDD) to define core gameplay mechanics and the game's overall theme.
  • Character Mechanics: Implemented C# scripts for Character Movement and the Combat System to ensure responsive control and interaction.
  • Item System: Developed an RNG System (Random Number Generation) to manage the spawning logic and probability of various egg types.
  • Enemy System: Programmed Enemy AI to autonomously track, chase, and consume Chickens and Bunnys using tag-based pathfinding logic.
  • Buff & Debuff System: Designed a dynamic modifier system where specific units (Guardian Bunnies) apply stacking buffs to global game parameters.
  • Audio System: Engineered the Audio Manager and settings UI to allow for real-time volume adjustment (SFX and Music).

Languages: C#

Tools: Unity , Visual Studio 2022
